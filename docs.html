Introduction

Welcome to the WorkforceOS technical documentation. WorkforceOS is a next-generation remote workforce tracking and analytics agent designed to give managers complete visibility into remote team productivity.

By utilizing lightweight background processes, WorkforceOS runs seamlessly on employee machines to capture vital work metrics without disrupting the workflow.

Key Features

  • timer

    Automated Time Tracking

    The agent automatically records active work hours, helping organizations maintain accurate timesheets effortlessly.

  • screenshot_monitor

    Smart Screenshots

    Takes periodic screenshots of the active monitor to visually document work progress. Dual monitor setups are supported natively.

  • apps

    Activity Logging

    Monitors active window titles and application usage to categorize productive vs non-productive time.

  • snooze

    Idle Detection

    Detects periods of inactivity (no mouse or keyboard input) to automatically pause tracking, ensuring accurate work logs.

Installation Guide

1. Download

Navigate to the homepage and click the Download Agent button to get the latest .exe installer for Windows.

2. Install

Run the downloaded setup file. Follow the on-screen instructions. Administrator privileges may be required to install the tracking services.

3. Authenticate

Launch WorkforceOS from your Start Menu. Enter the authentication token or credentials provided by your organization administrator to link your device.

Privacy & Security

WorkforceOS is built with security as a foundational principle. All captured data, including screenshots and keystroke telemetry, is encrypted in transit and at rest using AES-256 encryption.

Data is transmitted securely over HTTPS directly to your organization's designated cloud storage or on-premise servers. WorkforceOS does not store any sensitive data on third-party public clouds without your explicit configuration.
